# eSmart Attendance Privacy Policy

This repository contains the **Privacy Policy** for the **eSmart Attendance** app, which is used by students and universities (Badya, NUB, Memphis, etc.) to record attendance via:

1. **QR Code Scanning** using the device camera
2. **Location Verification** (Latitude & Longitude)
3. **Wi-Fi Access Point MAC Address**

The privacy policy is implemented as a **responsive HTML page** (`index.html`) suitable for web viewing and Google Play Store requirements.

---

## File Overview

- **index.html**: The main Privacy Policy page.
  - Uses **Roboto** font for clean, readable text.
  - Gradient header with eSmartSoft branding.
  - Card-style content container with clear headings and sections.
  - Mobile-friendly responsive design.
  - Sections include:
    - Introduction
    - Information We Collect
    - How We Use Your Information
    - Data Sharing
    - Data Security
    - Your Rights
    - Changes to This Policy
    - Contact Information

---

## How to Use

1. Copy the `index.html` file to your web server or app assets folder.
2. Ensure it is accessible via a URL (e.g., `https://yourdomain.com/privacy.html`).
3. Link to this URL in your **Google Play Store listing** under Privacy Policy.
